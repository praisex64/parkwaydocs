## Request parameters
<ParamField body="toAccountNumber" type="number" required>
  Customer Wallet Account Number to be Funded
</ParamField>

<ParamField body="amount" type="number" required>
  Customer Wallet Account Number to be Funded
</ParamField>

<ParamField body="memo" type="string" required>
  Transfer Memo (Comment)
</ParamField>

<ParamField body="transactionRef" type="string" required>
  Custom Transaction Reference
</ParamField>
