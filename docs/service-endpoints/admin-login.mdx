---
title: "Retrieve Bearer Token"
description: "This endpoint is called to get a bearer token to authenticate all API calls"
api: "POST /rc/rest/agent/login"
contentType: "application/x-www-form-urlencoded"
authMethod: "none"
---

<Tip>Ensure Content-Type is set to application/x-www-form-urlencoded</Tip>

All API calls except this one requires a Bearer token, to recieve this token the
integrating partner is expected to provide credentials issued for sandbox Testing
kindly reach out to us on [here](parkway.ng/get-demo) if you do not have this credentials

## Request Parameters

<ParamField body="userName" type="string" required placeholder="000001">
  Agent ID/Wallet ID of the partner usually numeric 6 or 10 digits
</ParamField>

<ParamField body="password" type="string" placeholder="5e884898da28047151d0e56f8dc6292773603" required>
  Agents Password, if you have received a clear password, this parameter would
  take the SHA256 hash of the password recieved.
</ParamField>

<ParamField body="sessionLength" type="number" optional placeholder="30">
  Token validity length in minutes, default is 30 minutes
</ParamField>

## Response Headers

<ResponseField name="Authorization" type="string">
  The bearer token for authentication
</ResponseField>

<ResponseField name="PWK" type="string">
  PWK
</ResponseField>

<ResponseField name="SessionID" type="string">
  Session ID token
</ResponseField>
