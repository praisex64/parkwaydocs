---
title: "Activate Customer Wallet"
description: "This endpoint is utilized immediately after creating the customer wallet which is inactive with the otp sent to the customer via sms to activate the wallet."
api: "POST /api/provider/wallets/activate"
contentType: "application/json"
---

<Tip>
  For Test environment please use any random 6 digit Number. On Production,
  Customer would recieve a 6 digit OTP
</Tip>

<RequestExample>

```json Request
{
  "accountNumber": "5000001766",
  "otp": "552587"
}
```
