---
title: "Search/Filter Created Wallet"
description: "This endpoint is utilized to search wallets you have created (Including the owner information of the wallet). it comes with filters to streamline your search"
api: "GET /api/provider/wallets/search"
---

## Body parameters

<ParamField query="pageSize" type="number" default="20" optional>
  it's Advisable to keep it at a max 20
</ParamField>


## Query Parameters
<ParamField query="page" type="number" placeholder="1" default="1" optional>
  You can skip by page, Default page is 1 (first page)
</ParamField>

<ParamField
  query="status"
  type="string"
  default="active"
  placeholder="active"
  optional
>
  This can either be active or inactive
</ParamField>

<ParamField
  query="accountNumber"
  type="string"
  placeholder="8441037152"
  optional
>
  Filter by specific Account Number or likeness e.g 8441037152 or 844103 (This will search for all wallet with this prefix)
</ParamField>
